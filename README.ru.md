# Zapret-Deck

Самостоятельный плагин Decky Loader для Steam Deck, объединяющий обход DPI через **Zapret** (nfqws) и безопасный VPN-туннель через **Cloudflare WARP (протокол MASQUE)** — обход интернет-блокировок и замедлений (включая YouTube и Discord) прямо в игровом режиме, без сторонних приложений.

> **Примечание:** Плагин полностью автономен. Не требует Amnezia VPN, warp-cli или других GUI-приложений. Все бинарники включены внутри плагина.

![Скриншот](assets/screenshot.png)

## 📋 Возможности

- **Два режима обхода**: Переключайтесь между **Zapret (Обход DPI)** для прямого доступа без замедлений (лучший выбор для YouTube 4K и Discord) и **WARP VPN (MASQUE)** для полного VPN-туннеля.
- **Протокол MASQUE**: WARP подключается через MASQUE (HTTP/3 поверх QUIC), что неотличимо от обычного HTTPS-трафика. Обходит блокировки WireGuard, применяемые провайдерами в России, без AmneziaWG.
- **Автоподбор стратегии**: Zapret тестирует и подбирает наилучшие параметры обхода DPI автоматически.
- **Менеджер блокируемых доменов**: Добавляйте и управляйте доменами для обхода прямо из интерфейса плагина.
- **100% автономность**: Работает в пространстве пользователя. Не требует компиляции ядра, отключения read-only файловой системы SteamOS или dkms.

## 📥 Установка

1. Скачайте последний релиз (`zapret-deck.zip`) со страницы [Releases](https://github.com/rosakodu/zapret-deck/releases).
2. Переместите ZIP-архив на ваш Steam Deck.
3. Включите **Режим разработчика** в системных настройках Steam Deck.
4. В настройках Decky Loader включите режим разработчика и выберите «Установить плагин из файла» (или перетащите ZIP).

## 🚀 Как использовать

### 1. Zapret (Обход DPI)
Выберите **Zapret (Обход DPI)** в выпадающем меню режимов плагина.
- Нажмите **Автоподбор стратегии** для автоматического тестирования и применения лучшей стратегии для вашей сети.
- Используйте **Редактировать список доменов** для добавления конкретных доменов (например, `youtube.com`).

### 2. WARP VPN (MASQUE)
Выберите **WARP (MASQUE VPN)** в выпадающем меню режимов плагина.
- При первом запуске нажмите **Сгенерировать аккаунт WARP** для регистрации нового клиентского профиля.
- Плагин автоматически подключится через MASQUE (HTTP/3) и установит зашифрованный туннель.
