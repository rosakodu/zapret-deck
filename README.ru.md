# Zapret-Deck

[🇬🇧 English](README.md) | [🇷🇺 Русский](README.ru.md) | [🇺🇦 Українська](README.uk.md)

Самостоятельный плагин для Decky Loader на Steam Deck, который объединяет в себе функции локального обхода DPI через **Zapret** (nfqws) и защищенного VPN-туннеля через **Cloudflare WARP (протокол MASQUE)** для обхода блокировок и замедлений (включая YouTube и Discord) прямо в игровом режиме.

> **Примечание:** Этот плагин на 100% автономен и не требует установки сторонних приложений (таких как Amnezia VPN, warp-cli или GUI-менеджеров рабочего стола) на ваш Steam Deck. Все бинарные файлы поставляются внутри плагина.

## 📋 Возможности

- **Двойная защита**: Выбирайте между **Zapret (Обход DPI)** для прямого скоростного доступа без VPN (лучший выбор для YouTube 4K / Discord) или **WARP VPN (MASQUE)** для полной маскировки трафика.
- **Протокол MASQUE**: Подключение к WARP осуществляется по современному протоколу MASQUE (HTTP/3 поверх QUIC). Этот трафик маскируется под обычные HTTPS-запросы, благодаря чему он не блокируется ТСПУ в РФ (в отличие от стандартного WireGuard).
- **Автоподбор стратегий**: Zapret содержит встроенный алгоритм автоподбора, который автоматически тестирует и выбирает наилучшие параметры десинхронизации пакетов для вашего провайдера.
- **Редактор списков сайтов**: Управляйте списком заблокированных доменов прямо из интерфейса плагина в игровом режиме.
- **100% Автономность**: Работает на уровне пользователя. Не требует отключения защиты файловой системы SteamOS (read-only), компиляции модулей ядра или настройки dkms.

## 📥 Установка

1. Скачайте последнюю версию (`zapret-deck.zip`) со страницы [Releases](https://github.com/rosakodu/zapret-deck/releases).
2. Скопируйте ZIP-архив на ваш Steam Deck.
3. Включите **Режим разработчика** в системных настройках Steam Deck.
4. В настройках Decky Loader перейдите в раздел разработчика, активируйте **Developer mode** и выберите «Установить плагин из файла» (или перетащите ZIP в область загрузки).

## 🚀 Использование

### 1. Zapret (Обход DPI)
Выберите режим **Zapret (Обход DPI)** в выпадающем списке плагина.
- Нажмите **Автоподбор стратегии**, чтобы автоматически протестировать и применить рабочую стратегию обхода для вашей сети.
- Используйте кнопку **Редактировать список сайтов**, чтобы добавить конкретные домены, которые необходимо обходить (например, `youtube.com`).

### 2. WARP VPN (MASQUE)
Выберите режим **WARP (MASQUE VPN)** в выпадающем списке плагина.
- При первом запуске нажмите **Сгенерировать WARP аккаунт** для создания профиля.
- Плагин автоматически подключится через MASQUE (HTTP/3) и поднимет защищенный туннель.
