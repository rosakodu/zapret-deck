إضافة لـ Decky Loader على Steam Deck تتيح لك تجاوز فحص الحزم العميق DPI (مثل YouTube و Discord) باستخدام استراتيجيات `zapret` (`nfqws`)، والاتصال بـ **Cloudflare WARP VPN** عبر `usque` بالتوجيه الشامل التلقائي.

![Screenshot](assets/screenshot.png)

## 📋 المميزات الرئيسية

- **الكشف التلقائي عن الاستراتيجية بنقرة واحدة**: خوارزمية ذكية تفحص الاتصال وتختار الاستراتيجية الأكثر فعالية خلال ثوانٍ (`ALT`, `FAKE`, `SPLIT`).
- **تبديل الاستراتيجيات بسهولة**: قائمة تفاعلية مع إمكانية إلغاء التحديد والتغيير اليدوي.
- **دمج Cloudflare WARP VPN**: تسجيل تلقائي لحساب WARP وتوجيه شامل للشبكة (تغيير IP وتجاوز جميع الحظر).
- **واجهة Decky Loader أصلية**: تصميم نظيف ومتناسق تمامًا مع قائمة Steam Deck.
- **التشغيل التلقائي واستعادة الحالة**: استعادة حالة الخدمة تلقائيًا بعد إعادة تشغيل الجهاز أو التنقل بين الأوضاع.
- **سرعة عالية واستهلاك منخفض**: يعمل على مستوى النواة عبر `nfqws` و `nftables` بدون استهلاك للمؤشر أو البطارية.
