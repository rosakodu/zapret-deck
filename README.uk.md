# Zapret-Deck

[🇬🇧 English](README.md) | [🇷🇺 Русский](README.ru.md) | [🇺🇦 Українська](README.uk.md)

Самостійний плагін для Decky Loader на Steam Deck, який об'єднує в собі функції локального обходу DPI через **Zapret** (nfqws) та захищеного VPN-тунелю через **Cloudflare WARP (протокол MASQUE)** для обходу блокувань та уповільнень (включаючи YouTube та Discord) прямо в ігровому режимі.

> **Примітка:** Цей плагін на 100% автономний і не вимагає встановлення сторонніх додатків (таких як Amnezia VPN, warp-cli або GUI-менеджерів робочого столу) на ваш Steam Deck. Всі бінарні файли постачаються всередині плагіна.

## 📋 Можливості

- **Подвійний захист**: Вибирайте між **Zapret (Обхід DPI)** для прямого швидкісного доступу без VPN (найкращий вибір для YouTube 4K / Discord) або **WARP VPN (MASQUE)** для повного маскування трафіку.
- **Протокол MASQUE**: Підключення до WARP здійснюється за сучасним протоколом MASQUE (HTTP/3 поверх QUIC). Цей трафік маскується під звичайні HTTPS-запити, завдяки чому він не блокується ТСПЗ у РФ (на відміну від стандартного WireGuard).
- **Автопідбір стратегій**: Zapret містить вбудований алгоритм автопідбору, який автоматично тестує та вибирає найкращі параметри десинхронізації пакетів для вашого провайдера.
- **Редактор списків сайтів**: Керуйте списком заблокованих доменів прямо з інтерфейсу плагіна в ігровому режимі.
- **100% Автономність**: Працює на рівні користувача. Не вимагає відключення захисту файлової системи SteamOS (read-only), компіляції модулів ядра або налаштування dkms.

## 📥 Встановлення

1. Завантажте останню версію (`zapret-deck.zip`) зі сторінки [Releases](https://github.com/rosakodu/zapret-deck/releases).
2. Скопіюйте ZIP-архів на ваш Steam Deck.
3. Увімкніть **Режим розробника** в системних налаштуваннях Steam Deck.
4. У налаштуваннях Decky Loader перейдіть до розділу розробника, активуйте **Developer mode** та виберіть «Встановити плагін з файлу» (або перетягніть ZIP в область завантаження).

## 🚀 Використання

### 1. Zapret (Обхід DPI)
Виберіть режим **Zapret (Обхід DPI)** у випадаючому списку плагіна.
- Натисніть **Автопідбір стратегії**, щоб автоматично протестувати та застосувати робочу стратегію обходу для вашої мережі.
- Використовуйте кнопку **Редагувати список сайтів**, щоб додати конкретні домени, які необхідно обходити (наприклад, `youtube.com`).

### 2. WARP VPN (MASQUE)
Виберіть режим **WARP (MASQUE VPN)** у випадаючому списку плагіна.
- При першому запуску натисніть **Згенерувати WARP акаунт** для створення профілю.
- Плагін автоматично підключиться через MASQUE (HTTP/3) та підніме захищений тунель.
